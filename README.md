# Apex GitHub Digest

Daily, concise summaries of your GitHub repository's open pull requests and issues, delivered straight to your inbox. Stay on top of your team's development without the noise.

### Features:
- Daily email summaries
- Supports multiple repositories
- Simple, secure GitHub OAuth integration

### How it works:
1. Connect your GitHub account.
2. Select the repositories you want to monitor.
3. Receive a clean, easy-to-read digest every morning.
